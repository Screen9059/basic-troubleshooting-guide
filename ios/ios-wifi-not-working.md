# Problem: WiFi Not Working on iOS Devices

## Devices Affected
- iPhone
- iPad

## Symptoms
- Cannot connect to WiFi
- Intermittent connection drops
- No internet access even when connected
- Slow WiFi speeds

## Possible Causes
- Incorrect WiFi password
- Router or network issues
- iOS software bug
- Network settings misconfiguration
- Interference from other devices

## Step-by-Step Solution
1. Toggle WiFi off and on (Settings → WiFi)  
2. Forget the WiFi network and reconnect  
3. Restart the iOS device  
4. Restart or reset the router  
5. Update iOS to the latest version (Settings → General → Software Update)  
6. Reset network settings (Settings → General → Transfer or Reset iPhone → Reset → Reset Network Settings)  

## Result
- WiFi connects normally  
- Stable internet access  

## Notes
- Tested on iOS 14, 15, 16
