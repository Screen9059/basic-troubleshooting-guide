# Problem: iOS Devices Running Slowly

## Devices Affected
- iPhone
- iPad

## Symptoms
- Apps take longer to open
- Slow response when switching apps
- Laggy scrolling or touch input
- Battery drains faster than usual

## Possible Causes
- Too many background apps running
- Full storage
- Outdated iOS version
- Software bugs
- Aging hardware

## Step-by-Step Solution
1. Restart the device  
2. Close unnecessary background apps (swipe up to close apps)  
3. Check storage (Settings → General → iPhone Storage) and free up space  
4. Update iOS to the latest version (Settings → General → Software Update)  
5. Clear app caches if possible  
6. Optional: Factory reset if performance does not improve (backup first)  

## Result
- Device runs smoothly  
- Apps open and respond faster  

## Notes
- Tested on iOS 14, 15, 16
