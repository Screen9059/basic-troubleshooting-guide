# Problem: Windows Computer Running Slowly

## Devices Affected
- Windows PC or Laptop

## Symptoms
- Slow app launches
- Lag when switching windows
- High CPU or memory usage
- Programs freeze or respond slowly

## Possible Causes
- Too many background programs running
- Low storage space
- Malware or viruses
- Outdated Windows version
- Old or failing hardware

## Step-by-Step Solution
1. Restart the computer
2. Check storage (Settings → System → Storage) and free up space
3. Close unnecessary background programs (Ctrl + Shift + Esc → Task Manager → End tasks)
4. Scan for malware (Windows Defender or antivirus)
5. Update Windows (Settings → Update & Security → Windows Update)
6. Optional: Upgrade hardware (more RAM, SSD)

## Result
- Windows PC runs smoothly
- Apps launch faster and multitasking works efficiently

## Notes
- Tested on Windows 10 and 11
