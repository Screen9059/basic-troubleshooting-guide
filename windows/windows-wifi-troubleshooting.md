# Problem: Windows Cannot Connect to WiFi

## Devices Affected
- Windows PC or Laptop

## Symptoms
- WiFi not connecting
- Limited or no internet access
- Drops connection frequently
- Cannot detect available networks

## Possible Causes
- Incorrect WiFi password
- Router issues
- Network driver problems
- Firewall or antivirus blocking connection
- Weak signal or interference

## Step-by-Step Solution
1. Restart the computer and router
2. Forget the WiFi network and reconnect with correct password
3. Update WiFi drivers (Device Manager → Network adapters → Update driver)
4. Disable firewall or antivirus temporarily to test
5. Run Windows Network Troubleshooter (Settings → Network & Internet → Status → Network Troubleshooter)
6. Check signal strength and move closer to the router
7. Optional: Reset network settings (Settings → Network & Internet → Status → Network reset)

## Result
- WiFi connects successfully
- Internet works reliably

## Notes
- Tested on Windows 10 and 11
